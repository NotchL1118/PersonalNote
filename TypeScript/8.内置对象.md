# 内置对象

JavaScript 中有很多[内置对象](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects)，它们可以直接在TypeScript中当做定义好了的类型。

## ECMAScript的内置对象

**`Boolean`、Number、`string`、`RegExp`、`Date`、`Error`**

类型和构造函数名一致

```ts
let b: Boolean = new Boolean(1)
let n: Number = new Number(true)
let s: String = new String('哔哩哔哩关注小满zs')
let d: Date = new Date()
let r: RegExp = /^1/
let e: Error = new Error("error!")
let xhr:XMLHttpRequest = new XMLHttpRequest()
```

### Promise

如果我们不指定返回的类型，TS无法推断返回的是什么类型

![img](https://pic.lsyfighting.cn/img/watermark,type_d3F5LXplbmhlaQ,shadow_50,text_Q1NETiBAcXExMTk1NTY2MzEz,size_20,color_FFFFFF,t_70,g_se,x_16.png)

所以需要我们指定返回的类型，Promise<类型>

```ts
function promise():Promise<number>{
   return new Promise<number>((resolve,reject)=>{
       resolve(1)
   })
}
 
promise().then(res=>{
    console.log(res)
})
```
